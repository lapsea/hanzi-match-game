# 汉字对对碰

一款面向小学低年级（6-9岁）的汉字词语消除游戏。

## 玩法

棋盘为 6×6 方格，共 36 个汉字随机摆放。点击两个格子，若它们能按顺序组成一个词语（如「苹」+「果」= 苹果），则消除这两个格子。全部消除即为过关。

- 点击**提示**：高亮一个可消除的词对
- 点击**重新开始**：重新打乱当前关卡
- 顺序反了会提示「换个顺序试试～」

## 关卡

内置 3 个难度级别，共 15 关：

| 级别 | 主题 | 关卡数 |
|------|------|--------|
| 第1级·启蒙 | 天地自然、动物、身体、家庭、植物 | 5关 |
| 第2级·进阶 | 食物、交通、学校、四季、职业 | 5关 |
| 第3级·挑战 | 自然现象、情绪、品德、地理、时间 | 5关 |

完成当前关自动解锁下一关。

## 自定义字库

在关卡选择页切换到「自定义」Tab，上传一个 `.txt` 文件即可生成自定义关卡，永久保存在本地浏览器中。

**txt 格式**（每行一个词语，至少 18 个）：
```
苹果
香蕉
天空
月亮
...
```

## 字库外置

关卡数据存放在 `dist/levels.json`，部署后直接在服务器上编辑该文件即可更新关卡内容，无需重新编译。

## 本地开发

```bash
npm install
npm run dev
```

修改 `public/levels.json` 后刷新浏览器即可看到关卡变化。

## 构建部署

```bash
npm run build
```

将 `dist/` 目录部署到任意静态文件服务器即可。

## Docker 部署

确保已安装 Docker，克隆仓库后执行：

```bash
docker compose up --build
```

启动后访问 `http://localhost:8080`。

如不使用 docker compose，也可手动构建：

```bash
docker build -t hanzi-math .
docker run -p 8080:80 hanzi-math
```
